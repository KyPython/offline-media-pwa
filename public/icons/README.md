# Icons

Place your PWA icons here. Required sizes:

- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192)
- icon-384.png (384x384)
- icon-512.png (512x512)

You can generate icons from a single source image using tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://www.appicon.co/

For now, you can use placeholder images or create simple colored squares for testing.
